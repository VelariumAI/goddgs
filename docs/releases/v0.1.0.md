## goddgs v0.1.0

Production-ready initial public release of `github.com/velariumai/goddgs`.

### Highlights

- DDG-first no-key search client with resilient fallback behavior.
- Typed provider failover engine (`ddg`, `brave`, `tavily`, `serpapi`).
- Production HTTP service (`goddgsd`) with health, readiness, and metrics endpoints.
- CLI runtime (`goddgs`) with `search`, `providers`, and `doctor` commands.
- Structured diagnostics, typed errors, and event hooks.
- Prometheus metrics, including circuit-breaker and fallback telemetry.
- Anti-bot resilience improvements:
  - fresh VQD on retries,
  - solver retries that do not burn retry budget,
  - circuit breaker with fail-fast behavior.

### Quality & Reliability

- Total test coverage gate enforced at **>=85.0%** in CI.
- CI includes vet, tests, race tests, staticcheck, and govulncheck.
- Release workflow publishes tagged binaries.

### Notes

- DDG endpoints used by the no-key provider are unofficial and may change.
- For critical production traffic, configure official provider keys for failover.
