## goddgs v0.1.1

Public release aligned with repository `main` after remote-history reconciliation.

### Includes

- Production DDG client with resilient retry/fallback behavior.
- Provider failover engine and typed diagnostics.
- CLI + HTTP service runtimes.
- Optional solver integrations for challenge workflows.
- Prometheus metrics and operational runbook.
- CI quality gates with enforced total coverage threshold (`>=85.0%`).

### Reliability Notes

- Challenge and anti-bot outcomes are environment-dependent.
- Official providers should be configured for reliability-critical fallback.
